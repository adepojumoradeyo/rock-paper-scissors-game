, # Rock Paper Scissors Game

A simple interactive Rock Paper Scissors game built with HTML, TAILWIND CSS, and JavaScript.

## How to Play

- Click Rock, Paper, or Scissors
- Computer will choose randomly
- Winner is displayed
- Scores update automatically
- Click reset to start over

## Technologies Used

- HTML
- TAILWIND CSS
- JavaScript

## Features

- Player vs Computer
- Score tracking
- Responsive design
- Reset button
